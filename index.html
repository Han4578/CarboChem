Welcome to CarboChem!

Here, you can build any carbon based chemical structures your want!

Reset Button
Resets the game.

Blue lines
Indicate where you can add an element, click on them to add one.

Element Button
Switch elements to add

Bond Button
Change number of bonds to add

Auto Button
Adds a hydrogen element onto every blue line.
Note: Be patient as it'll always be faster than manually clicking all of it.

Delete Button
Toggles delete mode.
In delete mode, the outermost elements and the last carbon in the carbon chain are marked as red. Click on them to delete it.

Zoom
Click on the + and - to change element size

Rules
Carbons can have 4 bonds.
Oxygens can have 2 bonds.
Hydrogen and halogens can have 1 bond.

Shortcut keys
C - Carbon
H - Hydrogen
O - Oxygen
L - Chlorine
B - Bromine
I - Iodine
R - Reset
A - Auto
D - Delete
1 - Single Bond
2 - Double Bond
3 - Triple Bond

Naming
When the structure is complete, its name will be automatically generated. This currently only applies to:
Alkane
Alkene
Alkyne
Alcohol
Carboxylic acid
*Combination of different functional groups also works, but not all

Excluded names:
2 or more oxygen atoms forming bonds together
Oxygen forming bonds with halogens
*They exist but were excluded due to being non-carbon based structures

A general guide on how to choose the main carbon chain
The main carbon chain is determined using these steps in sequence:
  1. It has the most number of substituents corresponding to the suffix (OH for alcohol, COOH for carboxylic acid, etc)
  2. It is the longest chain regardless of double or triple bonds
  3. It has the most double and triple bonds
  4. It has the most double bonds
  5. It has the lowest double or triple bond number
  6. It has the lowest double bond number
  7. It has the most branches (chloro-, methyl-, etc)
  8. It has the lowest branch number
